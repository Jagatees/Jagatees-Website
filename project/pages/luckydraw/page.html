Chrimstas Lucky Draw System

1 Jan 2024

Technology stack

"Embarking on my recent project, a unique Christmas-themed lucky draw system in Unity, has been a journey of immense learning and self-discovery. As I undertook this project solo, each step presented both a challenge and an opportunity to expand my skills and knowledge.

One of the most intriguing aspects I encountered was the implementation of a dual-display system. Learning how to synchronize two different monitors to create an interactive experience was not only technically challenging but also immensely rewarding. This feature allowed users to select an option on one screen and witness a corresponding animation on the other, thereby enhancing the user engagement.

The cornerstone of this project was developing the lucky draw mechanism. It required a deep dive into Unity's user interface capabilities. Designing clickable boxes and ensuring their responsiveness was a task that tested my patience and precision. The thrill of seeing these boxes function as intended, triggering animations seamlessly, was unparalleled.

Speaking of animations, integrating them into Unity was another significant learning curve. Animating the Christmas-themed elements, such as gift boxes and festive decorations, required a keen eye for detail and an understanding of Unity’s Animator and Animation tools. The process was like bringing a static scene to life, infusing it with the magic of Christmas.

What stood out the most was the realization of how much creativity and technical knowledge go hand in hand in game development. Balancing aesthetic appeal with functional design, especially in a festive theme like Christmas, involved a lot of artistic considerations coupled with technical executions.

This project also honed my problem-solving skills. Each hurdle, be it in design, animation, or functionality, was a puzzle that needed solving. It pushed me to explore Unity’s capabilities more deeply and think outside the box to find solutions.

Moreover, the project was a testament to the power of self-learning. Navigating through various aspects of Unity, from UI design to multi-screen setups, was a learning experience that went beyond any conventional tutorial. It was a blend of research, trial and error, and a lot of coding.

In conclusion, this project was not just about building a Christmas-themed lucky draw system in Unity. It was about pushing my boundaries, learning new skills, and discovering the joy in bringing an idea to fruition. It has left me with a deeper understanding of game development, a stronger set of skills, and an insatiable desire to keep learning and creating."
